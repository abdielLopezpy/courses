Infografía de JavaScript Básico

Este documento presenta algunos de los conceptos básicos de JavaScript, junto con ejemplos y casos de uso comentados en el código.

Variables

En JavaScript, las variables se utilizan para almacenar datos. Puedes declarar una variable usando var, let o const. 


var nombre = 'Juan';
console.log(nombre);  // Salida: Juan


let edad = 25;
console.log(edad);  // Salida: 25


const ciudad = 'Madrid';
console.log(ciudad);  // Salida: Madrid
Desafío: Declara tres variables usando var, let y const y muestra sus valores en la consola.

Tipos de Datos

JavaScript soporta varios tipos de datos, incluidos números, cadenas de texto, booleanos, objetos y arrays. 

let numero = 10;  // Número
let texto = 'Hola Mundo';  // Cadena de texto
let booleano = true;  // Booleano
let objeto = { nombre: 'Juan', edad: 25 };  // Objeto
let array = [1, 2, 3, 4, 5];  // Array
Desafío: Declara una variable de cada tipo de dato (número, cadena de texto, booleano, objeto y array) y muestra sus valores en la consola.

Operadores

Los operadores en JavaScript se utilizan para realizar operaciones en variables y valores. 

let a = 10;
let b = 5;

let suma = a + b;  // Suma
let resta = a - b;  // Resta
let multiplicacion = a * b;  // Multiplicación
let division = a / b;  // División
console.log(suma, resta, multiplicacion, division);  // Salida: 15, 5, 50, 2
Desafío: Declara dos variables y realiza operaciones de suma, resta, multiplicación y división, mostrando los resultados en la consola.

Funciones

Las funciones en JavaScript se utilizan para definir bloques de código que pueden ser llamados y ejecutados en cualquier momento. 

function saludar(nombre) {
    return 'Hola, ' + nombre;
}
console.log(saludar('Juan'));  // Salida: Hola, Juan
Desafío: Define una función que tome un parámetro y retorne un saludo. Llama a esta función y muestra el resultado en la consola.

Eventos

JavaScript puede responder a eventos del usuario, como clics, teclas presionadas, y más.
Desafío: Añade un manejador de eventos a un botón que muestre una alerta cuando se haga clic en él.
