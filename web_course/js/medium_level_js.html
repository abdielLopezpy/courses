Este documento presenta conceptos más avanzados de JavaScript, junto con ejemplos y casos de uso comentados en el código.

Funciones Avanzadas

JavaScript soporta funciones avanzadas como funciones flecha y funciones de alto orden. 

// Función flecha
const suma = (a, b) => a + b;

// Función de alto orden
function operacionMatematica(a, b, operacion) {
    return operacion(a, b);
}

console.log(operacionMatematica(5, 3, suma));  // Salida: 8
Desafío: Define una función flecha que calcule el cuadrado de un número y utilízala en una función de alto orden que procese una lista de números.

Manipulación del DOM

JavaScript permite manipular el DOM para modificar elementos HTML dinámicamente. 

// Obtener un elemento por su ID
const titulo = document.getElementById('titulo');

// Modificar el contenido del elemento
titulo.textContent = 'Nuevo Título';

// Crear un nuevo elemento y añadirlo al DOM
const parrafo = document.createElement('p');
parrafo.textContent = 'Este es un nuevo párrafo';
document.body.appendChild(parrafo);
Desafío: Modifica el contenido de un elemento HTML existente y crea un nuevo elemento para ser añadido al documento.

Eventos Avanzados

JavaScript también permite utilizar técnicas avanzadas de eventos como la delegación de eventos y eventos personalizados. 

// Delegación de eventos
document.addEventListener('click', function(event) {
    if (event.target.classList.contains('boton')) {
        console.log('Botón clicado');
    }
});

// Eventos personalizados
const miEvento = new CustomEvent('miEvento', { detail: { mensaje: 'Evento personalizado' } });
document.dispatchEvent(miEvento);
document.addEventListener('miEvento', function(event) {
    console.log(event.detail.mensaje);
});
Desafío: Utiliza delegación de eventos para manejar clics en varios botones y crea un evento personalizado con un mensaje.
