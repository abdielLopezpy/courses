Infografía de JavaScript Avanzado

Este documento presenta conceptos avanzados de JavaScript, incluyendo clases, módulos, manejo de errores, y uso de APIs modernas.

Clases y Módulos

JavaScript soporta la definición de clases y el uso de módulos para organizar y reutilizar código de manera eficiente. 

// Definición de una clase en JavaScript
class Persona {
    constructor(nombre) {
        this.nombre = nombre;
    }

    saludar() {
        return `Hola, soy ${this.nombre}`;
    }
}

// Uso de módulos para exportar e importar clases
// persona.js
export default Persona;

// main.js
import Persona from './persona.js';

const persona = new Persona('Juan');
console.log(persona.saludar());  // Salida: Hola, soy Juan
Desafío: Define una clase avanzada con métodos y utilízala en un módulo separado para importarla en tu archivo principal.

Manejo de Errores

JavaScript permite manejar errores de manera efectiva usando el bloque try-catch para capturar excepciones. 

try {
    // Bloque de código que puede lanzar una excepción
    throw new Error('Este es un error personalizado');
} catch (error) {
    // Captura y manejo de la excepción
    console.error('Error:', error.message);
}
Desafío: Crea una función que intencionalmente lance una excepción y utiliza try-catch para manejarla.

Uso de APIs Modernas

JavaScript puede interactuar con APIs modernas, como la API Fetch para realizar peticiones HTTP asincrónicas. 

// Ejemplo de uso de la API Fetch para obtener datos de un servidor
fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error al obtener datos:', error));
Desafío: Utiliza la API Fetch para realizar una petición GET a una API pública y muestra los resultados en tu página.
