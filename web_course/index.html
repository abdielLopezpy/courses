Introducción a HTML, CSS y JavaScript

En esta guía, aprenderás los conceptos básicos de HTML, CSS y JavaScript con ejemplos prácticos. HTML (HyperText Markup Language) se utiliza para crear la estructura de una página web, CSS (Cascading Style Sheets) se utiliza para estilizar y diseñar la página web, y JavaScript se utiliza para añadir interactividad.

HTML Básico

HTML utiliza "etiquetas" para estructurar el contenido. Aquí tienes algunas etiquetas HTML básicas:

Encabezados (<h1> a <h6>)

Los encabezados se utilizan para títulos y subtítulos, desde el más importante (<h1>) hasta el menos importante (<h6>).

<h1>Título Principal</h1>
<h2>Subtítulo 1</h2>
<h3>Subtítulo 2</h3>

Título Principal

Subtítulo 1

Subtítulo 2

CSS Básico

CSS se utiliza para estilizar y diseñar una página web. Aquí tienes algunas propiedades CSS básicas:

Color

La propiedad color se utiliza para cambiar el color del texto.

p {
    color: blue; /* El texto será azul */
}

Este texto es azul.

Fondo

La propiedad background-color se utiliza para cambiar el color de fondo de un elemento.

div {
    background-color: yellow; /* El fondo será amarillo */
}

Este div tiene un fondo amarillo.

JavaScript Básico

JavaScript se utiliza para añadir interactividad a una página web. Aquí tienes un ejemplo básico:

Botón Interactivo

En este ejemplo, usaremos JavaScript para mostrar un mensaje cuando se haga clic en un botón.

<button class="btn" onclick="mostrarMensaje()">Haz clic aquí</button>

<script>
function mostrarMensaje() {
    alert('¡Hola! Has hecho clic en el botón.');
}
</script>
