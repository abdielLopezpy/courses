Infografía Intermedia de HTML

Esta infografía presenta etiquetas HTML de nivel intermedio, con ejemplos y desafíos para los estudiantes.

<table> - Tabla

Las tablas se utilizan para organizar datos en filas y columnas.

<table>
    <tr>  <!-- Fila -->
        <th>Encabezado 1</th>  <!-- Encabezado de columna -->
        <th>Encabezado 2</th>
    </tr>
    <tr>  <!-- Fila -->
        <td>Dato 1</td>  <!-- Celda de datos -->
        <td>Dato 2</td>
    </tr>
</table>
Desafío: Crea una tabla con 3 columnas y 3 filas de datos.

<form> - Formulario

Los formularios se utilizan para recolectar datos del usuario.

<form action="procesar.php" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">  <!-- Campo de texto -->
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">  <!-- Campo de email -->
    <br>
    <input type="submit" value="Enviar">  <!-- Botón de enviar -->
</form>
Desafío: Crea un formulario con campos para nombre, email y un campo de selección.

<video> - Video

La etiqueta <video> se utiliza para insertar videos.

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">  <!-- Fuente de video -->
    Tu navegador no soporta la etiqueta de video.
</video>
Desafío: Inserta un video con controles y añade una descripción alternativa.

<audio> - Audio

La etiqueta <audio> se utiliza para insertar audio.

<audio controls>
    <source src="audio.mp3" type="audio/mp3">  <!-- Fuente de audio -->
    Tu navegador no soporta la etiqueta de audio.
</audio>
Desafío: Inserta un archivo de audio con controles.

<section> - Sección

La etiqueta <section> se utiliza para definir secciones dentro del contenido.

<section>
    <h2>Título de la Sección</h2>  <!-- Uso: Encabezado de la sección -->
    <p>Contenido de la sección.</p>  <!-- Uso: Contenido dentro de la sección -->
</section>
Desafío: Crea una sección con un título y un párrafo de contenido.

<article> - Artículo

La etiqueta <article> se utiliza para contenido independiente, como un artículo o una entrada de blog.

<article>
    <h2>Título del Artículo</h2>  <!-- Uso: Encabezado del artículo -->
    <p>Contenido del artículo.</p>  <!-- Uso: Texto del artículo -->
</article>
Desafío: Crea un artículo sobre un tema de tu interés.

<nav> - Navegación

La etiqueta <nav> se utiliza para contener enlaces de navegación.

<nav>
    <ul>
        <li><a href="#inicio">Inicio</a></li>  <!-- Uso: Enlace de navegación -->
        <li><a href="#servicios">Servicios</a></li>  <!-- Uso: Otro enlace de navegación -->
    </ul>
</nav>
Desafío: Crea una barra de navegación con enlaces a diferentes secciones.

<footer> - Pie de Página

La etiqueta <footer> se utiliza para definir el pie de página de un documento o sección.

<footer>
    <p>© 2024 Mi Sitio Web</p>  <!-- Uso: Información del pie de página -->
</footer>
Desafío: Añade un pie de página con información de contacto.
