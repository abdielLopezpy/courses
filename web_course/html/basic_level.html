Infografía de Etiquetas HTML Principales

Este documento presenta algunas de las etiquetas HTML más utilizadas, junto con ejemplos y casos de uso comentados en el código.

<h1> a <h6> - Encabezados

Los encabezados son utilizados para definir títulos y subtítulos, con <h1> siendo el más importante y <h6> el menos importante. 

<h1>Título Principal</h1>  
<h2>Subtítulo 1</h2>        
<h3>Subtítulo 2</h3>        
<h4>Subtítulo 3</h4>        
<h5>Subtítulo 4</h5>        
<h6>Subtítulo 5</h6>
Desafío: Crea una estructura de página con un título principal y varios subtítulos.

<p> - Párrafo

La etiqueta <p> se utiliza para definir párrafos. 

<p>Este es un párrafo de ejemplo.</p>
Desafío: Añade tres párrafos de texto sobre cualquier tema.

<a> - Enlace

La etiqueta <a> se utiliza para definir hipervínculos. 

<a href="https://www.ejemplo.com">Visita Ejemplo</a>  
<a href="pagina.html">Enlace Relativo</a>
Desafío: Crea enlaces a tres sitios web diferentes.

<img> - Imagen

La etiqueta <img> se utiliza para insertar imágenes. 

<img src="imagen.jpg" alt="Descripción de la imagen">
Desafío: Inserta una imagen con una descripción alternativa adecuada.

<ul>, <ol>, <li> - Listas

Las etiquetas <ul> y <ol> se utilizan para listas no ordenadas y ordenadas, respectivamente. <li> define cada elemento de la lista. 

<ul> 
    <li>Elemento 1</li>  
    <li>Elemento 2</li>  
</ul>

<ol> 
    <li>Elemento 1</li>  
    <li>Elemento 2</li>  
</ol>
Desafío: Crea una lista no ordenada con al menos cinco elementos y una lista ordenada con tres elementos.

<div> - División

La etiqueta <div> se utiliza para agrupar y estructurar otros elementos en un contenedor. 

<div>
    <p>Contenido dentro de una división.</p>  
</div>
Desafío: Agrupa varios párrafos en una división.

<section> - Sección

La etiqueta <section> se utiliza para definir secciones dentro del contenido. 

<section>
    <h2>Título de la Sección</h2>  
    <p>Contenido de la sección.</p>  
</section>
Desafío: Crea una sección con un título y contenido relevante.

<article> - Artículo

La etiqueta <article> se utiliza para contenido independiente, como un artículo o una entrada de blog. 

<article>
    <h2>Título del Artículo</h2>  
    <p>Contenido del artículo.</p>  
</article>
Desafío: Escribe un artículo corto sobre un tema de tu elección.

<nav> - Navegación

La etiqueta <nav> se utiliza para contener enlaces de navegación. 

<nav>
    <ul>
        <li><a href="#inicio">Inicio</a></li>  
        <li><a href="#servicios">Servicios</a></li>  
    </ul>
</nav>
Desafío: Crea un menú de navegación con enlaces a diferentes secciones de la página.

<footer> - Pie de Página

La etiqueta <footer> se utiliza para definir el pie de página de un documento o sección. 

<footer>
    <p>© 2024 Mi Sitio Web</p>  
</footer>
Desafío: Añade un pie de página con información de contacto.
