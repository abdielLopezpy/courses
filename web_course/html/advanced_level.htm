Infografía Avanzada de HTML

Esta infografía presenta etiquetas HTML avanzadas, con ejemplos y desafíos para los estudiantes.

<canvas> - Canvas

La etiqueta <canvas> se utiliza para dibujar gráficos y animaciones mediante JavaScript.

<canvas id="miCanvas" width="300" height="200">
    Tu navegador no soporta la etiqueta de canvas.
</canvas>
<script>
    var canvas = document.getElementById('miCanvas');
    var ctx = canvas.getContext('2d');
    ctx.fillStyle = 'blue';
    ctx.fillRect(10, 10, 150, 100);
</script>
Desafío: Dibuja un círculo en el canvas.

<svg> - SVG

La etiqueta <svg> se utiliza para definir gráficos vectoriales.

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>
Desafío: Crea un gráfico SVG con varias formas geométricas.

<template> - Plantilla

La etiqueta <template> se utiliza para definir fragmentos de contenido reutilizable que pueden ser clonados e insertados en el DOM.

<template id="miPlantilla">
    <p>Este es un párrafo dentro de una plantilla.</p>
</template>
<script>
    var plantilla = document.getElementById('miPlantilla');
    var clon = plantilla.content.cloneNode(true);
    document.body.appendChild(clon);
</script>
Desafío: Crea una plantilla que incluya una lista de elementos.

<picture> - Imagen Adaptable

La etiqueta <picture> se utiliza para definir una imagen adaptable que cambia según el tamaño de la pantalla o el dispositivo.

<picture>
    <source media="(min-width: 650px)" srcset="img_grande.jpg">
    <source media="(min-width: 465px)" srcset="img_mediana.jpg">
    <img src="img_pequena.jpg" alt="Imagen adaptable">
</picture>
Desafío: Implementa una imagen adaptable con tres diferentes tamaños.

Microdata - Datos Enriquecidos

Utiliza microdata para mejorar el SEO de tu página web añadiendo información estructurada.

<div itemscope itemtype="http://schema.org/Person">
    <span itemprop="name">John Doe</span>
    <div itemprop="address" itemscope itemtype="http://schema.org/PostalAddress">
        <span itemprop="streetAddress">123 Main St</span>
        <span itemprop="addressLocality">Springfield</span>
        <span itemprop="addressRegion">IL</span>
        <span itemprop="postalCode">62701</span>
    </div>
    <span itemprop="telephone">(555) 555-5555</span>
</div>
Desafío: Añade microdata a un perfil de producto con nombre, descripción y precio.

<details> y <summary> - Detalles y Resumen

Las etiquetas <details> y <summary> se utilizan para crear secciones de contenido que el usuario puede expandir y contraer.

<details>
    <summary>Más Información</summary>
    <p>Aquí hay más detalles que se pueden mostrar u ocultar.</p>
</details>
Desafío: Crea una sección de preguntas frecuentes usando detalles y resumen.

<iframe> - Marco en Línea

La etiqueta <iframe> se utiliza para incrustar otro documento HTML dentro de la página actual.

<iframe src="https://www.ejemplo.com" width="600" height="400">
    Tu navegador no soporta iframes.
</iframe>
Desafío: Incrusta un video de YouTube usando un iframe.

<meta> - Metadatos

La etiqueta <meta> se utiliza para definir metadatos sobre el documento HTML, como la descripción, palabras clave y autor.

<meta name="description" content="Esta es una descripción de mi página web.">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="Tu Nombre">
Desafío: Añade metadatos a tu documento HTML.
