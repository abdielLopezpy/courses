Formulario Básico HTML

Este formulario básico en HTML cubre los elementos esenciales para crear un formulario funcional.

<input> - Campo de Entrada

El campo de entrada básico se utiliza para recibir datos del usuario.

<label for="name">Nombre:</label>
<input type="text" id="name" name="name" required>
Desafío: Añade un campo de entrada para el apellido.

<input type="email"> - Correo Electrónico

El campo de correo electrónico valida automáticamente que el formato del email sea correcto.

<label for="email">Correo Electrónico:</label>
<input type="email" id="email" name="email" required>
Desafío: Añade un campo de entrada para el teléfono.

<textarea> - Área de Texto

El área de texto permite al usuario ingresar múltiples líneas de texto.

<label for="message">Mensaje:</label>
<textarea id="message" name="message" rows="4" required></textarea>
Desafío: Limita el número de caracteres a 200.

<button> - Botón de Envío

El botón de envío se utiliza para enviar el formulario.

<button type="submit">Enviar</button>
Desafío: Añade un botón para restablecer el formulario.
