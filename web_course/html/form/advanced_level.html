Infografía Avanzada de Formularios HTML

Esta infografía presenta elementos y atributos avanzados para formularios HTML, con ejemplos y desafíos para los estudiantes.

<datalist> - Lista de Datos

La etiqueta <datalist> proporciona una lista de opciones predefinidas para un campo de entrada.

<label for="navegador">Elige tu navegador:</label>
<input list="navegadores" id="navegador" name="navegador">
<datalist id="navegadores">
    <option value="Chrome">
    <option value="Firefox">
    <option value="Safari">
    <option value="Edge">
</datalist>
Desafío: Crea un campo de entrada con una lista de datos para seleccionar lenguajes de programación.

<output> - Resultado

La etiqueta <output> representa el resultado de un cálculo.

<form oninput="result.value=parseInt(a.value)+parseInt(b.value)">
    <input type="range" id="a" value="50"> +
    <input type="number" id="b" value="50"> =
    <output name="result" for="a b">100</output>
</form>
Desafío: Crea un formulario que calcule el área de un triángulo en base a la base y altura proporcionadas por el usuario.

<progress> - Barra de Progreso

La etiqueta <progress> representa el progreso de una tarea en curso.

<label for="file">Subiendo archivo: </label>
<progress id="file" value="32" max="100"> 32% </progress>
Desafío: Implementa una barra de progreso que se actualice automáticamente usando JavaScript.

Validación de Formularios

HTML5 proporciona varios atributos para la validación de formularios, como required, pattern, min, max, y más.

<form>
    <label for="email">Correo Electrónico:</label>
    <input type="email" id="email" name="email" required>
    <br>
    <label for="password">Contraseña:</label>
    <input type="password" id="password" name="password" pattern=".{8,}" title="Ocho o más caracteres" required>
    <br>
    <button type="submit">Enviar</button>
</form>
Desafío: Crea un formulario de registro que valide el correo electrónico, la contraseña y que la confirmación de contraseña coincida con la contraseña ingresada.

Entrada de Fechas

HTML5 introduce nuevos tipos de entrada para fechas y horas: date, time, datetime-local, month, y week.

<form>
    <label for="birthday">Fecha de Nacimiento:</label>
    <input type="date" id="birthday" name="birthday">
    <br>
    <label for="appt">Hora de la Cita:</label>
    <input type="time" id="appt" name="appt">
    <br>
    <button type="submit">Enviar</button>
</form>
Desafío: Crea un formulario que solicite la fecha y hora de un evento y que valide que la fecha no sea anterior a la fecha actual.

Subida de Archivos

La etiqueta <input type="file"> se utiliza para permitir a los usuarios seleccionar y subir archivos.

<form action="/upload" method="post" enctype="multipart/form-data">
    <label for="file">Elige un archivo:</label>
    <input type="file" id="file" name="file">
    <br>
    <button type="submit">Subir</button>
</form>
Desafío: Crea un formulario que permita la subida de múltiples archivos y muestre una vista previa de los mismos antes de subirlos.
