Formulario Intermedio HTML

Este formulario de nivel intermedio en HTML utiliza validaciones y elementos avanzados para proporcionar una experiencia robusta al usuario.

Nombre de Usuario

El campo para el nombre de usuario utiliza un patrón para asegurar un mínimo de 5 caracteres.

<label for="username">Nombre de Usuario:</label>
<input type="text" id="username" name="username" pattern=".{5,}" title="Cinco o más caracteres" required>
Desafío: Añade una validación para que el nombre de usuario no contenga caracteres especiales.

Correo Electrónico

El campo de correo electrónico valida automáticamente el formato del email.

<label for="email">Correo Electrónico:</label>
<input type="email" id="email" name="email" required>
Desafío: Añade un mensaje personalizado que se muestre si el formato del correo electrónico no es válido.

Contraseña

El campo de contraseña asegura un mínimo de 8 caracteres mediante el uso de un patrón.

<label for="password">Contraseña:</label>
<input type="password" id="password" name="password" pattern=".{8,}" title="Ocho o más caracteres" required>
Desafío: Añade validaciones adicionales para requerir al menos un número y un carácter especial.

Confirmar Contraseña

Este campo verifica que la confirmación de la contraseña coincida con la contraseña ingresada.

<label for="confirmPassword">Confirmar Contraseña:</label>
<input type="password" id="confirmPassword" name="confirmPassword" required>
Desafío: Implementa la lógica para comparar y validar que ambas contraseñas coincidan usando JavaScript.

Lista de Datos

La etiqueta <datalist> proporciona una lista de lenguajes de programación predefinidos.

<label for="programmingLanguages">Lenguajes de Programación:</label>
<input list="languages" id="programmingLanguages" name="programmingLanguages">
<datalist id="languages">
    <option value="Python">
    <option value="JavaScript">
    <option value="Java">
    <option value="C++">
    <option value="Ruby">
</datalist>
Desafío: Añade más lenguajes de programación a la lista y valida que el campo no quede vacío.

Fecha de Nacimiento

El campo de fecha asegura que el usuario seleccione una fecha de nacimiento.

<label for="birthday">Fecha de Nacimiento:</label>
<input type="date" id="birthday" name="birthday" required>
Desafío: Valida que la fecha seleccionada no sea futura.

Biografía

El campo de texto permite al usuario ingresar una breve biografía.

<label for="bio">Biografía:</label>
<textarea id="bio" name="bio" rows="4" required></textarea>
Desafío: Añade una validación para limitar el número de caracteres a 500.
