Diferentes formas de incluir CSS en HTML

Existen tres formas principales de incluir CSS en un documento HTML:

  • CSS Externo
  • CSS Interno
  • CSS En-línea

CSS Externo

El CSS externo se coloca en un archivo separado con la extensión .css. Este archivo se enlaza al documento HTML usando la etiqueta <link> dentro del <head>. 

<!-- Archivo HTML -->
<head>
    <link rel="stylesheet" href="styles.css">
</head>

<!-- Archivo CSS (styles.css) -->
body {
    background-color: #f4f4f4;
    font-family: Arial, sans-serif;
}

Este ejemplo muestra cómo enlazar un archivo CSS externo.

CSS Interno

El CSS interno se coloca dentro de la etiqueta <style> dentro del <head> del documento HTML. 

<head>
    <style>
        body {
            background-color: #f4f4f4;
            font-family: Arial, sans-serif;
        }
    </style>
</head>

Este ejemplo muestra cómo incluir CSS directamente en el documento HTML usando la etiqueta <style>.

CSS En-línea

El CSS en-línea se coloca directamente dentro de un elemento HTML usando el atributo style. 

<p style="color: blue; background-color: yellow;">
    Este es un párrafo con estilo en-línea.
</p>

Este es un párrafo con estilo en-línea.
