Infografía de Propiedades CSS Principales

Este documento presenta algunas de las propiedades CSS más utilizadas, junto con ejemplos y casos de uso comentados en el código.

color, background-color - Colores

Las propiedades color y background-color se utilizan para definir los colores del texto y del fondo, respectivamente. 

.elemento {
    color: #333;  /* Uso: Color del texto */
    background-color: #f4f4f4;  /* Uso: Color de fondo */
}
Desafío: Cambia el color del texto y el fondo de un elemento.

font-family, font-size, font-weight - Fuentes

Las propiedades font-family, font-size y font-weight se utilizan para definir la apariencia del texto. 

.elemento {
    font-family: Arial, sans-serif;  /* Uso: Tipo de fuente */
    font-size: 16px;  /* Uso: Tamaño de la fuente */
    font-weight: bold;  /* Uso: Grosor de la fuente */
}
Desafío: Aplica diferentes estilos de fuente a un párrafo.

border - Bordes

La propiedad border se utiliza para definir los bordes de un elemento. 

.elemento {
    border: 1px solid #333;  /* Uso: Borde sólido de 1 píxel */
}
Desafío: Añade bordes a un contenedor.

margin, padding - Márgenes y Rellenos

Las propiedades margin y padding se utilizan para el espacio exterior e interior de un elemento. 

.elemento {
    margin: 20px;  /* Uso: Espacio exterior */
    padding: 10px;  /* Uso: Espacio interior */
}
Desafío: Ajusta los márgenes y el relleno de un elemento.

display: flex - Flexbox

La propiedad display: flex se utiliza para crear un diseño flexible. 

.contenedor {
    display: flex;  /* Uso: Contenedor flexible */
    justify-content: center;  /* Uso: Centrar contenido horizontalmente */
    align-items: center;  /* Uso: Centrar contenido verticalmente */
}
Desafío: Crea un contenedor flexible con elementos centrados.

display: grid - Grillas

La propiedad display: grid se utiliza para crear un diseño de grillas. 

.contenedor {
    display: grid;  /* Uso: Contenedor de grilla */
    grid-template-columns: 1fr 1fr;  /* Uso: Dos columnas iguales */
    gap: 20px;  /* Uso: Espacio entre elementos */
}
Desafío: Crea una grilla con dos columnas.
