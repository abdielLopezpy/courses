Infografía Avanzada de Propiedades CSS

Este documento presenta propiedades avanzadas de CSS, incluyendo animaciones, transiciones y el modelo de cajas, con ejemplos prácticos y desafíos.

transition - Transiciones

La propiedad transition se utiliza para suavizar los cambios de estilos de los elementos. 

.elemento {
    transition: all 0.3s ease-in-out;  /* Uso: Suaviza todos los cambios de estilo en 0.3 segundos */
}

.elemento:hover {
    background-color: #ffc107;  /* Cambio de color de fondo al pasar el ratón */
}
Desafío: Añade una transición suave a un botón.

animation - Animaciones

La propiedad animation se utiliza para crear animaciones CSS. 

@keyframes ejemplo {
    from { transform: scale(1); }
    to { transform: scale(1.5); }
}

.elemento {
    animation: ejemplo 1s infinite alternate;  /* Uso: Animación de escalado infinito */
}
Desafío: Crea una animación que cambie el tamaño de un elemento.

box-sizing - Modelo de Cajas

La propiedad box-sizing se utiliza para controlar el tamaño y el espaciado de los elementos. 

.elemento {
    box-sizing: border-box;  /* Uso: Incluye el padding y el borde en el tamaño total del elemento */
    width: 100%;
    padding: 20px;
    border: 2px solid #333;
}
Desafío: Ajusta el tamaño de un elemento utilizando el modelo de cajas.

Flexbox Avanzado

Propiedades avanzadas de flexbox para un control más detallado del diseño flexible. 

.contenedor {
    display: flex;
    flex-wrap: wrap;  /* Uso: Permite a los elementos flexibles envolver a una nueva línea */
    gap: 20px;
}

.elemento {
    flex: 1 1 200px;  /* Uso: Elementos flexibles con un tamaño mínimo de 200px */
}
Desafío: Crea un contenedor flexible avanzado con elementos envolventes.

Grillas Avanzadas

Propiedades avanzadas de grid para un diseño de grillas más complejo. 

.contenedor {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));  /* Uso: Columnas adaptables con un tamaño mínimo */
    gap: 20px;
}

.elemento {
    grid-column: span 2;  /* Uso: Elemento que ocupa dos columnas */
}
Desafío: Crea una grilla avanzada con elementos que ocupen múltiples columnas.
